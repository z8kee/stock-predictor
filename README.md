# 📈 AI Stock Predictor & Live Trading Dashboard

A real-time, machine-learning-powered stock analysis dashboard. This application uses a hybrid CNN-LSTM neural network to predict short-term price movements, integrates live news sentiment analysis using FinBERT, and features a custom Autoencoder to detect "Black Swan" market anomalies.

## ✨ Features

* **Real-Time Data Polling:** Fetches live price action and historical candlestick data via the Yahoo Finance API.
* **Smooth Charting Engine:** Integrated TradingView Lightweight Charts for a professional, interactive UI.
* **Deep Learning Predictions:** Uses a trained CNN-LSTM model to generate 'BUY', 'SELL', or 'HOLD' probabilities.
* **Live Sentiment Analysis:** Scrapes the latest financial news and processes the headlines through Hugging Face's `ProsusAI/finbert` to scale prediction probabilities dynamically.
* **Black Swan Detection:** Utilizes an Autoencoder neural network to calculate real-time Mean Absolute Error (MAE) reconstruction scores. 

## 🛠️ Tech Stack

* **Backend:** Python 3.11, Flask, Gunicorn
* **Machine Learning:** TensorFlow / Keras (CNN-LSTM & Autoencoder), Scikit-Learn
* **NLP / Sentiment:** Hugging Face `transformers` (FinBERT)
* **Frontend:** HTML5, CSS3, Vanilla JavaScript, TradingView Lightweight Charts
* **Deployment:** Docker

## 🚀 Getting Started (Docker)

1. Clone the repository:
   `git clone https://github.com/yourusername/stock-predictor.git`
   `cd stock-predictor`
2. Build the Docker image:
   `docker build -t stock-predictor .`
3. Run the container:
   `docker run -p 5000:5000 stock-predictor`
4. Open your browser and navigate to `http://localhost:5000`.
